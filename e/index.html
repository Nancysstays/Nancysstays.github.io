About This Application

This application demonstrates the use of the MACD indicator for algorithmic trading. It fetches historical stock data, calculates the MACD, and provides buy/sell signals based on the analysis.

The application is built using HTML, CSS, and JavaScript, with Plotly.js for charting and the Alpha Vantage API for data retrieval.
